# 胡辣汤品牌推广HTML演示文稿

## 项目概述
这是一个基于HTML5的响应式演示文稿，用于胡辣汤品牌推广与文化传播。

## 技术特性
- **响应式设计**: 支持桌面和移动设备
- **现代交互**: 平滑的页面切换动画
- **键盘导航**: 支持方向键和空格键控制
- **触摸支持**: 移动设备滑动操作
- **中英双语**: 符合国际化需求

## 设计风格
- **主色调**: 中国红(#C73E1A) + 鎏金色(#D4AF37)
- **辅助色**: 暖白色(#FFF8F0) + 深褐色(#3E2723)
- **字体**: Noto Serif SC (标题) + Noto Sans SC (正文)
- **动画**: 淡入淡出效果，营造传统韵味

## 页面结构
1. **封面页**: 品牌logo、标题和slogan
2. **品牌介绍**: 历史渊源和文化背景
3. **品牌故事**: 创立理念和发展历程
4. **产品展示**: 主要产品系列介绍
5. **精选食材**: 优质原料展示
6. **制作工艺**: 生产流程说明
7. **营养价值**: 健康功效介绍
8. **市场优势**: 竞争力分析
9. **目标群体**: 消费者画像
10. **发展愿景**: 品牌未来规划
11. **未来规划**: 具体发展策略
12. **联系方式**: 品牌联系信息

## 使用方法
1. 直接在浏览器中打开 `hulatang_presentation.html`
2. 使用以下方式导航：
   - 点击左右箭头按钮
   - 使用键盘方向键或空格键
   - 点击底部导航点
   - 移动设备上左右滑动

## 浏览器兼容性
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 自定义说明
- 修改颜色主题：调整CSS中的颜色变量
- 更新内容：编辑对应的HTML内容区域
- 添加新页面：复制现有slide结构并更新导航
