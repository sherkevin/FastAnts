# 胡辣汤宣传PPT需求文档

## 1. PPT主题
**“中原味道，一碗入魂：胡辣汤品牌文化之旅”**
- 旨在通过PPT全面展示胡辣汤的独特魅力，不仅将其定位为一道美食，更是一种文化符号和情感寄托。

## 2. 页面数量
**总计10页**
- 封面 (1页)
- 目录 (1页)
- 胡辣汤简介 (1页)
- 历史渊源与文化传承 (1-2页)
- 核心食材与匠心工艺 (2页)
- 品牌故事与独特优势 (1-2页)
- 健康价值与味蕾体验 (1页)
- 结尾与号召 (1页)

## 3. 风格要求
- **整体风格**：中国传统风格与现代简约设计相结合。
- **色调**：以暖色调为主，如中国红、陶土棕、米白，营造温暖、地道、有食欲的视觉感受。
- **字体**：标题使用有力量感的书法字体或宋体变体，正文使用清晰易读的黑体。
- **视觉元素**：
    - 融入水墨、祥云、青花瓷等中国传统纹样作为背景或装饰。
    - 大量使用高清、色彩饱和的胡辣汤成品图、食材特写图、制作过程图。
    - 图表设计需简洁明了，符合整体风格。

## 4. 重点内容
- **文化深度**：强调胡辣汤作为中原地区（尤其是河南）代表性早餐的历史地位和文化意义。
- **品质展示**：突出我们品牌胡辣汤的选材考究（如：优质牛肉、地道骨汤、纯天然香料）和传统工艺。
- **感官刺激**：通过生动的图片和描述性文字，让观众仿佛能闻到香气、尝到味道。
- **品牌差异化**：清晰阐述我们的胡辣汤与其他竞品相比的独特之处（例如：独家秘方、更健康的配方等）。

## 5. 特殊要求
- **图片质量**：所有使用的图片必须为高清原图，无水印，能够充分展现食物的诱人色泽。
- **动态效果**：希望在“匠心工艺”部分嵌入一个简短的（约15-30秒）视频，展示胡辣汤从熬汤到出锅的关键步骤。视频需有字幕和背景音乐。
- **多场景适用性**：PPT内容需具备一定的灵活性，能够稍作调整即可用于不同场合，如：招商会、美食文化节、线上社交媒体宣传等。
- **交付物**：除了PPT源文件外，还需提供一个PDF版本和一份包含所有图片、视频等素材的文件夹。
