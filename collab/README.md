# 胡辣汤宣传PPT使用说明

## 文件说明
- `hulatang_ppt.html` - 完整的HTML PPT文件，包含所有内容和功能

## 如何使用

### 1. 打开PPT
- 直接双击 `hulatang_ppt.html` 文件，用浏览器打开
- 或者在浏览器中拖拽文件到页面中

### 2. 导航方式
- **键盘导航**: 使用左右方向键 (← →) 切换页面
- **鼠标导航**: 点击页面任意位置切换到下一页
- **触屏导航**: 在移动设备上左右滑动切换页面

### 3. 功能特性
- **响应式设计**: 自动适配电脑、平板、手机屏幕
- **进度条**: 页面底部显示当前阅读进度
- **页码指示**: 右下角显示当前页码和总页数
- **背景音乐**: 右上角音乐按钮可开关背景音乐
- **地图功能**: 第8页包含推荐店铺的地图位置
- **动画效果**: 页面切换和内容显示带有流畅动画

### 4. 内容结构
1. **封面** - 胡辣汤主题介绍
2. **历史起源** - 600多年的发展历史
3. **主要食材** - 六种核心食材展示
4. **制作过程** - 四步制作流程
5. **特色亮点** - 四大特色介绍
6. **早餐对比** - 与其他早餐的对比分析
7. **文化内涵** - 胡辣汤的文化意义
8. **推荐店铺** - 三家知名店铺及地图位置
9. **联系方式** - 联系信息和二维码
10. **结束页** - 感谢页面

### 5. 技术要求
- 现代浏览器 (Chrome, Firefox, Safari, Edge)
- 网络连接 (用于加载地图功能)
- JavaScript 必须启用

### 6. 分享方式
- **在线分享**: 上传到网页服务器直接访问
- **离线分享**: 直接发送HTML文件给他人
- **演示模式**: 全屏模式下进行演示

### 7. 注意事项
- 首次打开可能需要加载地图资源，请耐心等待
- 背景音乐需要用户交互后才能播放
- 在移动设备上建议横屏观看以获得最佳体验

## 设计特点
- **新中式风格**: 传统元素与现代设计结合
- **暖色调配色**: 橙色、红色系营造温暖食欲感
- **双语内容**: 中英文对照，适合国际交流
- **交互体验**: 丰富的动画和交互效果
